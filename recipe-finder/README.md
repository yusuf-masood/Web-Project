# Recipe Finder App

## Project Overview
Recipe Finder is a web application that will allow users to browse, search, and view different recipes.  
The project is being developed as part of the Web Development final project.

The application will be built using:
- Angular for the frontend
- Django and Django REST Framework (DRF) for the backend

---

## Technologies Used

Frontend:
- Angular
- TypeScript
- HTML
- CSS

Backend:
- Django
- Django REST Framework (DRF)

Tools:
- GitHub
- Angular CLI
- Postman
